# Business-Performance-and-Forecasting-Analysis-in-Excel
📌 Overview

This project performs a comprehensive business performance analysis using a simulated transactional dataset. It evaluates revenue, costs, and profitability across time while applying analytical and forecasting techniques commonly used by business and financial analysts.

The model demonstrates how Microsoft Excel can be used to transform raw data into actionable insights through structured analysis, KPI development, forecasting, and scenario modelling.

Objectives
- To analyse company performance using key financial metrics such as revenue, profit, and margin
- To identify trends and seasonality in business performance over time
- To evaluate performance across different periods using structured analysis techniques
- To forecast future revenue using a 3-month moving average model
- To simulate business scenarios (e.g. cost increases, demand changes) and assess their impact on profitability

Prerequisites
- Microsoft Excel (2016 or later recommended)
- Basic understanding of:
- Excel formulas
- Pivot tables
- Charts

Key Features
- Structured raw dataset for realistic business analysis
- Data cleaning and transformation layer
- KPI calculations including revenue, profit, and margin
- Time-series forecasting using moving averages
- Scenario modelling for decision-making
- Interactive dashboard for visual insights

Workbook Structure
- Raw_Data – Original transactional dataset
- Data_Cleaning – Processed and structured data with calculated fields
- KPIs – Key performance indicators
- Analysis – Aggregated insights and breakdowns
- Forecast – Revenue projections using a 3-month moving average
- Scenario_Model – What-if analysis for business decisions
- Dashboard – Visual summary of performance metrics and trends
- Lists – Supporting data for dropdowns and validation

Forecasting Methodology

The forecast is based on a 3-month moving average model, where each period’s forecast is calculated as the average of the previous three months of actual revenue.

- Early periods are initialised using available data
- Future months display forecast values while actual revenue remains blank
- This approach provides a simple and interpretable time-series projection
- Scenario Analysis

The model includes a scenario tool that allows users to adjust:

- Cost increase (%)
- Demand growth (%)

The tool dynamically updates:

- Profit
- Profit margin

This illustrates how analysts assess the impact of business decisions under various assumptions.

Dashboard

The dashboard provides a clear and concise overview of business performance, including:

- Total revenue
- Total profit
- Profit margin
- Revenue trends over time
- Forecasted performance

It is designed to replicate an executive-level reporting view commonly used in business environments.
